# Design System

Direção visual:

- base preta;
- superfícies cinza quase preto;
- texto branco suave;
- vermelho discreto para destaque;
- bordas finas;
- cards com raio de 8px;
- animações mínimas.

Tokens iniciais vivem em `packages/ui/src/theme/tokens.ts`.

O desktop evita visual de landing page e abre direto na experiência de produto. A leitura rápida durante champion select é prioridade.
