# Documentação Sparta

Esta pasta reúne as decisões técnicas do MVP. A fonte principal do produto é simples: recomendar melhor antes da partida e revisar melhor depois, sem assistência durante a partida.

Leitura recomendada:

- `architecture.md` para visão geral.
- `setup.md` para ambiente local.
- `riot-compliance.md` para limites de integração.
- `scoring-model.md` e `draft-recommendation.md` para os motores iniciais.
- `database.md` para o schema inicial.
